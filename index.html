workflow

From raw rows to a 30‑day forecast

Each stage is a standalone script — run them in sequence, or let the dashboard call them for you.

01 / generate

Synthetic dataset

3 years of daily sales with trend, weekly/yearly seasonality, promotions, and holidays.

02 / engineer

Feature engineering

Date parts, lag features (1/7/14/30 days), and rolling mean & std windows.

03 / train

Model training

Linear Regression, Random Forest, and XGBoost compared on a held‑out 60‑day window.

04 / forecast

Recursive forecast

The best model predicts sales day‑by‑day, feeding each prediction back as the next lag.

evaluation
